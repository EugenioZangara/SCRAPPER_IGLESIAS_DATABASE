# Django SEO Content Skill

## Objective

Improve the visible content of Django pages for SEO and conversion.

## Review

Analyze:

- H1 uniqueness
- H2/H3 hierarchy
- Above-the-fold clarity
- Search intent
- Local SEO signals
- Service descriptions
- FAQs
- Internal links
- CTA clarity
- Thin content
- Duplicate content between pages

## Output format

For each page, return:

1. Current issue
2. Recommended H1
3. Recommended title tag
4. Recommended meta description
5. Suggested section structure
6. FAQ suggestions
7. Internal link opportunities
8. CTA improvement

## Rules

- Write for humans first.
- Avoid keyword stuffing.
- Keep brand tone professional and clear.
- Prioritize conversion as well as SEO.
- Respect the existing design and template structure.
