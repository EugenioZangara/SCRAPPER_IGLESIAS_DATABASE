# ParroGuía

**[parroguia.com](https://parroguia.com)** — Directorio de parroquias católicas de la Arquidiócesis de Buenos Aires.

Sistema para centralizar, gestionar y enriquecer información sobre parroquias: horarios de misa, eventos, redes sociales y datos de contacto.

## Stack

- **Backend**: Django 6.x, Python 3.13
- **Base de datos**: PostgreSQL
- **Frontend**: HTML/CSS + HTMX
- **IA**: Gemini (principal) / OpenRouter (fallback)
- **Deploy**: Render
